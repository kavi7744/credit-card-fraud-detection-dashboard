# credit-card-fraud-detection-dashboard

## Project Overview
This project analyzes credit card transaction data to identify fraud patterns

## Dataset
- Total Transactions: 1,048,575
- Fraud Transactions: 6006
- Fraud Rate: 0.57%

## Key Metrics
- Total Transactions
- Fraud Transactions
- Fraud Percentage
- Fraud Risk Score

## Dashboard Insights
- Fraud activity peaks during late-night hours
- Online shopping categories show higher fraud risk
- High-value transactions have increased fraud probability

## Tools Used
- Mysql
- python(pandas)
- Power BI
- Excel / CSV
- Data Analysis
